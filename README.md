# How To Run

- 해당 문제 경로에 터미널로 접근 후에 명령어로 구동(우클릭 -> 통합 터미널에서 열기)

  - cpp 파일은 코드러너로 우클릭 기동 또는 "ctrl + alt + c" 로 빌드 후, "ctrl + alt + r" 로 구동
  - java 파일은 `java Main.java`
  - js 파일은 `node Main.js`

- javascript 코딩 시, fs 모듈을 사용하기 위하여 `npm install fs` 필요함
- 코드의 자세한 형태는 example 참고
